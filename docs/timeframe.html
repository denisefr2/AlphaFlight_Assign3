Week Three

Project Developer

Marketing

Financial Advisor

Milestone

In this week, the team begins to work through the planning cycle (reference the Cat-A-Wall Development Cycle). As such the following is to occur:

- Project Developer: Write a blurb considering the target audience of the Cat-A-Wall game. Outlining if there is an age restriction or any other factors. This will be completed once the above is reflected in the assignment sheet.

- Marketing: Outline the processes for Alpha Flight to advertise the game. Additionally, show examples of merchandise that will be available for purchase to aid in funding the project. This will be considered completed once the above is reflected in the assignment sheet.

- Financial Advisor: Create a brief describing the costs that the project could incur including but not limited to software, hardware, training, and recruitment. This should be done in a manner that allows Alpha Flight to scale resources if funding is secured in the future. This task is considered completed once it is reflected in the assignment sheet.

Week Four

Video Game Writer

Video Game Writer

Platform Developer

Milestone

In this week, the team should complete the planning cycle of the Cat-A-Wall development cycle and move into the storyboard phase by completing the follow:

- Video Game Writer: Develop the main storyline and begin to branch out some of the missions. This process will focus on the first level as the project will work through developing a level at a time. This task is completed once the outline is added into the assignment sheet.

- Video Game Writer: Create a whole world map and point out any safe places, mission places etc. This phase will be completed when a first draft (it will more than likely evolve over the project) is added into the assignment document.

- Platform Developer: Begin to explore Unreal Engine and familiarise yourself with it. Provide a back brief on your experience. This task is purely designed to be part of the planning phase and as such is just research and does not require any outputs outside of documenting your experience in the assignment sheet.

Week Six

Game Designer

Game Designer

Game Designer

Milestone

Over the period of this week the team is to continue to build on the storyboard phase of the Cat-A-Wall development cycle this will include:

- Game designer: Begin to develop the scene visions creating a general theme that will be continued throughout the game. This scene will be created once 2D (2 Dimensional) visualisations of the cities and particulars are designed.

- Game designer: Develop the hub and race details. This is to include 2D designs to visualise the concept. Furthermore, each race and places of significance in the hub should include a description.

- Game designer: Develop the class and craft details. This is to include 2D designs to visualise the concept. Furthermore, each class and craft should include a description.

Week Seven

Level Designer

Project Manger

Cyber security Specialist

Milestone

By the end of the week Alpha Flight as a team should achieve the following:

- Level Designer: Begin to conduct specialised training as a level designer. Training should focus on C++, HTML, CSS, Java Script, and Unreal Engine.

- Project Manager: Run a team meeting focusing on the team’s progress thus far. Note any issues, successes, and points for improvement. Be reactive and flexible to change the timeline if required.

- Cyber Security Advisor: Start the development of a security policy for Cat-A-Wall. This is to include but not limited to how the team will host the information, and what protection methods will be incorporated into the game to protect the theft of information.

Project Manager

Project Manager

Milestone

The end of week fifteen signifies the end of the first timeframe. Therefore, in week fifteen it is important for the team to come together to contemplate the successes and failures of the project.

- At this point a complete consolidation of the previous fifteen weeks should be examined to better understand where the project is at.

- The team should also consider what to do next and reflect on whether the task should or should not be continued. If the plan should continue the next time cycle should be developed.
