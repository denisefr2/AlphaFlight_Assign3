Game Developer

Job Brief:

This position is responsible for building the game using objects created by our artists and audio team. The incumbent must be able to work closely with other team members to produce a quality end product.

The Game Developer will be responsible for following the design laid out by the Game Designer but must be able to contribute meaningfully where needed to the overall game design.

Reports to:

Game Designer

Education:

Degree in Computer Science, Software Engineering or Games Development

Skills:

Unreal Engine 4

Visual Studio 2019

Blueprint & C++ or other programming language (eg C, Java)

High level knowledge of APIs and libraries

Expert in on more or programming specialities (eg AI, 3D rendering 3D animation, physics or audio)

Ability to think creatively and contribute new designs and ideas

Strong analytical skills

Ability to work in a team

Passion for and knowledge of emerging gaming trends, current best practices and technologies

Problem solving using logic

Knowledge of sound and visual design

Experience:

Proven experience in full lifecycle game development

Ability to interpret design specifications into playable code

Experience animating characters and objects

Experience working with illustrators/sound designers as a team to produce a quality end product

Experience with code documentation and writing/contributing to technical documents
