Team comments on Tools

GitHub

Teams

Denise

GitHub is the industry default for storing and sharing code for good reason. The interface is relatively simple to learn and yet houses powerful tools that I feel these assignments barely scratches the surface of what’s possible.

I look forward to learning more of the usefulness and function of GitHub as both my study and career in IT progress in the future.

As with assignment 2, the tools used proved suitable to the task. The functions offered by Teams to both record and create a transcript of the meetings is useful if any need is found to revisit the content.

Using the video chat function for meetings creates a sense of connection that might otherwise be missing where team members are situated at geographically distant locations.

The ability to organise and save documents and drafts to created folders is important and a critical facet of keeping the team’s various tasks clear and precise.

Jennafer

GitHub proved to be a useful platform enabling teams to collaborate on code and build a website together effectively.

As the team had identified the design concept for the site in assignment two the development of this assignment was seamless.

As a team, we learnt from the points identified in assignment two and are proud of the final product.

As with assignment two, Alpha Flight has continued to rely heavily on Microsoft Teams for communication.

Our team chose to conduct two weekly syncs (Wednesday and Sunday) using the platform. These meetings were the backbone to planning tasks and communicating our ideas for the assignments.

Additionally, we maintained communication via the chat function in which everyone was able to convey to them what they were up to. The chat function was vital to keeping the team informed and engaged in the project between meetings.

Alpha Flight also continued the same file structure and storage system within Microsoft Teams for all of the documents in a method proven to be successful in assignment two.

Microsoft Teams was integral in maintaining Alpha Flight’s communication noting the dispersed geographic locations of the members.

 

Zhaohe

We used GitHub as a platform to build our website, once past the initial learning curve, our team has shown to be more proficient at using this platform.

Each team member’s work can be seen in GitHub’s audit trail, we were able to work on our parts of the website and merge all our work seamlessly.

Reflecting back on the upload history of Microsoft Teams files, it has shown a consistent effort of contribution by all team members. We have once again benefited greatly from this tool.

Our group was able to effectively communicate and commit our works efficiently.
